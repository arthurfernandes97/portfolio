## Sobre

Este repositório funciona como um ponto central para os projetos que desenvolvo ao longo da graduação em Ciência da Computação e da minha transição de carreira para a área de TI.

Aqui estão reunidos laboratórios práticos, scripts e anotações voltados principalmente para Redes de Computadores, Infraestrutura, GNU/Linux e Suporte Técnico. Cada projeto possui documentação própria, descrevendo o objetivo, o cenário, as configurações realizadas e os testes de validação.
